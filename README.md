# ☀️ Solar Energy Estimator

A Python application that estimates solar energy production based on solar panel specifications.

## Features

- Calculates installed power (kW)
- Calculates daily energy production
- Calculates monthly energy production
- Calculates yearly energy production
- Classifies production level (Low / Medium / High)

## Inputs

- Number of Solar Panels
- Power per Panel (W)
- Daily Sunlight Hours
- System Efficiency (%)

## Technologies

- Python
- Mathematical Calculations
- Conditional Statements (if / elif / else)

## Example

Installed Power: 5.50 kW

Daily Energy Production: 28.05 kWh

Production Level: MEDIUM
